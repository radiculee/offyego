## What's shipped (Phase 0 through Phase 4 Part 2a)

- Domain registered (offyego.ie + offyego.com defensive), repo on GitHub, deployed to Vercel.
- Scaffold: Next 16 + React 19 + TypeScript strict + Tailwind v4 + Vitest. Three gates (age, Ireland, location), 8-state useReducer machine, Overpass fetch, Roulette animation, RadiusSlider, SpinButton.
- Phase 3: PubCard (name, distance, walking time, address, open/closed), GuiltTripModal (fires at spin 5), PubMap (Leaflet, SSR-safe dynamic import), Spin Again + Get Directions.
- Phase 4 Part 2a: theme-neutral layout and typography foundation, CSS variable system, hydration-safe personality switching, voice routing audit.
- Build clean. 24/24 tests passing. All user-facing strings routed through voices.*.ts.

## What's verified

Full flow verified on localhost and in incognito at end of the previous session: age gate passes, location acquired, radius slider works, spin fetches real pubs via Overpass, result card renders with map + challenge + Get Directions + Spin Again, guilt modal fires at spin 5. Real-device testing (iPhone/Android) is still pending and is deferred to Phase 5.
